# 💸 Alke Wallet - Mi Prototipo de Billetera Digital

¡Hola! 👋 Te presento **Alke Wallet**, un proyecto que desarrollé con mucho entusiasmo durante mi aprendizaje como desarrollador Front-End. El desafío era crear desde cero la interfaz funcional de una billetera digital, y este es el resultado.

Este proyecto fue mi campo de juego para aplicar y profundizar mis conocimientos en HTML, CSS y JavaScript, además de aprender a integrar librerías tan potentes como Bootstrap y jQuery para construir una experiencia de usuario moderna y dinámica.

## ✨ Lo que puedes hacer en Alke Wallet

*   **🔐 Iniciar Sesión:** Un portal de acceso para entrar a tu billetera.
*   **💰 Gestionar tu Dinero:** Visualiza tu saldo, haz depósitos para recargar tu cuenta y envía dinero a quien quieras.
*   **💸 Enviar Dinero Fácilmente:** Un formulario de transferencias con una práctica función de autocompletar para que encuentres a tus contactos sin esfuerzo.
*   **📊 Historial de Movimientos:** Mantén un registro de todas tus transacciones. ¡Todo queda guardado en una tabla clara y fácil de leer!
*   **📱 Diseño Adaptable:** Ya sea en tu computadora o en tu celular, la interfaz se ajusta para que la uses cómodamente donde quieras.

## 🚀 ¿Quieres probarla?

¡Es muy fácil! No necesitas instalar complejas dependencias.

1.  Clona o descarga este repositorio en tu computadora.
2.  Entra en la carpeta `alke-wallet`.
3.  Abre el archivo `index.html` en tu navegador favorito. ¡Y a disfrutar!

## 🛠️ Las herramientas detrás del proyecto

*   **HTML5:** El esqueleto que da estructura y significado a todo.
*   **CSS3:** El arte para darle vida y un estilo único.
*   **Bootstrap 5:** Mi gran compañero para lograr un diseño profesional y responsivo de forma ágil.
*   **JavaScript (ES6+):** El cerebro que le da toda la interactividad y lógica a la aplicación, usando `localStorage` para simular la persistencia de datos.
*   **jQuery & jQuery UI:** Para añadir esas chispas de magia, como el autocompletado en la lista de contactos.
